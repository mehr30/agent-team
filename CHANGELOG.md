# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-02-04

### Added
- Initial release with 22 workflows
- Core roles: architect, security, frontend, backend, database, testing, devops, reviewer, docs
- Advanced workflows: github, scaffold, debug, refactor, optimize, migrate, monitor, dependencies, changelog, auto-docs, onboarding, emergency, complexity
- Orchestrator that combines all roles
- 2-pass critic loop for self-review
- CLAUDE.md shared knowledge base with file locking
- One-command installer via curl
- GitHub Actions CI
- Issue templates for bugs and features
- Contributing guide
